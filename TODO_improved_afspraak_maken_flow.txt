Current State Analysis:
Currently, your flow is User Request → Email → Manual Call Back → Calendar Negotiation → Confirmation. This creates friction and "phone tag."

Since you are running a clinic with specific time slots (Intake 90m vs Follow-up 60m), the best improvement is to remove the manual negotiation step or make the request much more specific.

Here are 3 ways to improve the flow, ranging from "Quick Fix" to "Professional Integration".

Option 1: The "Professional" Fix (Embed Calendly/Cal.com)
Instead of a contact form for appointments, embed a booking widget. This allows the client to pick a specific time that syncs with your Google/Outlook calendar.

Why: No back-and-forth emails. The client books, you get a notification, it's done.

How to implement:

Create a free account on Cal.com or Calendly.
Create two event types: "Intake (90min)" and "Vervolgbehandeling (60min)".
Update your components/blocks/contact-form.tsx to conditionally render the calendar when "Afspraak maken" is selected.

'use client';

import React, { useState, useEffect } from 'react';
import { Section } from '../layout/section';
import { tinaField } from 'tinacms/dist/react';
import { PageBlocksContactForm } from '../../tina/__generated__/types';
import { Button } from '../ui/button';
import Cal, { getCalApi } from "@calcom/embed-react"; // You would need: npm install @calcom/embed-react

export const ContactForm = ({ data }: { data: PageBlocksContactForm }) => {
    const [appointmentType, setAppointmentType] = useState('appointment');
    
    // Initialize Cal.com (optional, if you use their modal)
    useEffect(()=>{
      (async function () {
        const cal = await getCalApi({});
        cal("ui", {"styles":{"branding":{"brandColor":"#EB5A3C"}},"hideEventTypeDetails":false,"layout":"month_view"});
      })();
    }, [])

    return (
        <Section background={data.background!}>
            <div className="max-w-4xl mx-auto"> {/* Widen container for calendar */}
                {/* Title & Desc ... */}
                
                {/* Toggle Switch */}
                <div className="flex justify-center mb-8">
                    <div className="bg-gray-100 p-1 rounded-full inline-flex">
                        <button
                            onClick={() => setAppointmentType('appointment')}
                            className={`px-6 py-2 rounded-full text-sm font-medium transition-all ${
                                appointmentType === 'appointment' 
                                ? 'bg-white text-orange-600 shadow-sm' 
                                : 'text-gray-500 hover:text-gray-900'
                            }`}
                        >
                            Direct Afspraak Maken
                        </button>
                        <button
                            onClick={() => setAppointmentType('information')}
                            className={`px-6 py-2 rounded-full text-sm font-medium transition-all ${
                                appointmentType === 'information' 
                                ? 'bg-white text-orange-600 shadow-sm' 
                                : 'text-gray-500 hover:text-gray-900'
                            }`}
                        >
                            Vraag Stellen
                        </button>
                    </div>
                </div>

                {appointmentType === 'appointment' ? (
                    <div className="bg-white rounded-xl shadow-sm border overflow-hidden min-h-[600px]">
                        {/* Replace 'jeroen-hoff' with your actual cal.com username */}
                        <Cal 
                            calLink="jeroen-hoff"
                            style={{width:"100%",height:"100%",overflow:"scroll"}}
                            config={{layout: 'month_view'}}
                        />
                    </div>
                ) : (
                    <form className="max-w-xl mx-auto space-y-6">
                       {/* ... Your existing form fields for generic questions ... */}
                    </form>
                )}
            </div>
        </Section>
    );
};
